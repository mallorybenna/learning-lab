# ai-learning

A weekly practice for staying on the frontier of AI engineering: every ~week I ship one small, self-contained project that produces a real artifact. Projects rotate across four areas — AI-augmented dev tooling, AI features in web apps, agentic systems & evals, and AI infra & data — and are grounded in real work, current trends, or things I'm personally into.

Stack: TypeScript-first (Angular, NestJS, Nx), built against the latest Claude models.

# learning-lab

A landing ground for little learning-by-building projects — I ship one small, self-contained thing at a time and keep the artifact here. A steady stream of these are guided "arcs" from my daily learning routine (AI-relevant engineering, systems architecture, UI/UX), but anything I'm tinkering with is welcome to land here.

Stack: TypeScript-first (Angular, NestJS, Nx), built with the latest Codex/Claude models.

Guided projects live in [`arcs/<name>/`](arcs/) with their code, an `architecture.md` (Mermaid diagram), a short design-decisions (ADR) note, and a `meta.json`. The table below is **auto-generated** from those `meta.json` files — run `npm run build-readme`, or just `git commit` and the pre-commit hook rebuilds it.
